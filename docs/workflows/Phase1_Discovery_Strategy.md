# FASE 1: DISCOVERY & STRATEGY

## 🎯 Objetivo de la Fase
Validar la viabilidad del mercado, definir el alcance exacto del MVP, y establecer los perfiles de los usuarios y sus flujos de compra para la plataforma FUTX STORE.

## 📦 Entregables Específicos
1. Documento de Análisis de Mercado (TAM, SAM, SOM, Competencia).
2. Documento Scope del MVP (Priorización MoSCoW, Listado de Features, Riesgos).
3. Documento de User Journey (5 perfiles de usuario, Paint Points, Happy Paths).

## 🤖 Prompts para Ejecutar (Copiar y Pegar)

**Prompt 1.1: Validación de Mercado**
```text
@brainstorming Eres analista estratégico senior. Valida el mercado para FUTX STORE (plataforma D2C de camisetas de fútbol retro/vintage en España):
- ¿Es viable el mercado de camisetas retro?
- ¿Cuál es el TAM (Total Addressable Market) en España?
- ¿Qué competencia existe?
- ¿Cuáles son los principales riesgos?
- ¿Qué métricas definen éxito en el mes 1 (apuntando a >2000€ en ventas, 50-75 órdenes)?
Proporciona análisis cuantitativo y cualitativo estructurado.
```

**Prompt 1.2: MVP Scope Definition**
```text
@brainstorming Eres PM senior. Define scope MVP de FUTX STORE para 4 semanas con un equipo ágil (1-2 devs). 
Evalúa las siguientes features bajo priorización MoSCoW:
- Catálogo de productos (50+ SKUs), Carrito de compra, Checkout (Stripe), Email marketing automation, Admin panel, Analytics dashboard, CMS, SEO, Customer support chat.
Proporciona:
1. Features imprescindibles (Must), importantes (Should), deseables (Could), y Phase 2 (Won't).
2. Data model básico.
3. Tech stack recomendado.
```

**Prompt 1.3: User Journey Mapping**
```text
@brainstorming Define las 5 personas usuario principales para FUTX STORE:
1. El Coleccionista Serio (35-50, invierte 500€+/mes)
2. El Aficionado Nostálgico (40-60, compra ocasional)
3. El Hype Beast Joven (18-28, seguidor de tendencias)
4. El Inversor de Vintage (35-55, busca valor)
5. El Regalador Inteligente (25-40, ocasional pero alto ticket)

Para cada persona, detalla: Motivación de compra, Pain points, Buyer's journey completo desde descubrimiento hasta post-compra, y Tasa de conversión esperada.
```

## ✅ Checklist de Completitud
- [ ] Análisis de mercado finalizado y revisado.
- [ ] Scope definido (MoSCoW) y documentado.
- [ ] User personas creados.
- [ ] Estrategia de Marketing planificada.

## ⚠️ Riesgos y Mitigación
- **Riesgo:** Scope creep (intentar abarcar demasiadas funcionalidades en 4 semanas).
- **Mitigación:** Aplicar estrictamente el marco MoSCoW y limitarse al núcleo transaccional.
- **Riesgo:** Mercado saturado o competencia desleal con réplicas.
- **Mitigación:** Enfatizar la autenticidad y las garantías en la propuesta de valor.

- **Duración:** 2 Días (Semana 1)

- Día 2: User Journeys y planificación de Copywriting estratégico.
